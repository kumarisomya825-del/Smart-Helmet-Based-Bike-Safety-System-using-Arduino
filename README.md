Smart Helmet Based Bike Safety System using Arduino

Overview

This project improves rider safety by allowing the bike motor to run only when the rider wears the helmet and no alcohol is detected.

Features

- Helmet Detection using IR Sensor
- Alcohol Detection using MQ-3 Sensor
- Motor Control using Relay
- LCD Display Status
- Red / Green LED Indication
- Buzzer Alert

Components Used

- Arduino UNO
- IR Sensor
- MQ-3 Sensor
- Relay Module
- DC Motor
- LCD 16x2
- LEDs
- Buzzer
- Battery

Working

- Helmet detected + No alcohol = Motor ON
- Helmet not detected = Motor OFF
- Alcohol detected = Motor OFF + Buzzer ON
